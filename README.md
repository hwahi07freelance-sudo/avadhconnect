# Avadh Connect

A community directory app for Avadh Heliconia.

## Features
- **Residents**: Browse businesses by category, view details, and download catalogs.
- **Businesses**: Create accounts, manage listings, and track views.
- **Admins**: Review and approve listings with AI assistance.

## Quick Start
1.  Open `index.html` in your web browser.
2.  **To Test Business Flow**:
    - Go to Sign Up -> Business Owner.
    - Create an account.
    - Fill out the "My Listing" form.
3.  **To Test Admin Flow**:
    - Open `admin/index.html`.
    - Go to Listings.
    - Run "AI Check" and Approve the listing.
4.  **To Test Public Flow**:
    - Go back to `index.html`.
    - Browse the category you added the business to.
    - You should see your new listing!

## Tech Stack
- HTML5, CSS3 (Modern Variables), JavaScript (ES6+)
- `localStorage` for data persistence (Mock DB)
- Ready for Firebase integration

## Customization
- **Styles**: Edit `css/style.css` to change colors or fonts.
- **Config**: Update `js/firebase-config.js` with your Firebase credentials to go live.
